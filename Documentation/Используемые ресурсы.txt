Основной задачей данного проекта является реализация сортировки Шелла на языке Java в визуализированном виде.

Дополнительной и отдельной задачей проекта является работа с файлами Excel и составления отчета посещаемости 
студентов по данным документам. 
Для совмещения студенты групп из данных файлов сортируются в алфавитном порядке при помощи сортировки Шелла.

Данный проект написан при помощи следующих программных средств:

Среда разработки - IDE Intellij Idea community edition версии 15.0.3
Набор Java разработки - JDK-8u91
Framework для упрощенной разработки интерфейса - JavaFX Scene Builder версии 2.0
Система контроля версий - Git версии 2.7.3

Написано на операционной системе Windows 7

Ноутбук на базе Intel(R) Core(TM) i3-3217U CPU @ 1.80GHz

Проект загружен на репозиторий GitHub - https://github.com/MisterAleks/Shell_Sort

ВАЖНОЕ ПРИМЕЧАНИЕ по Java FX Scene Builder:

Начиная с Oracle Java SE 8u40, Oracle не предоставляет отдельный набор сопутствующих JavaFX Scene Builder бинарных файлов
Эти версии JavaFX Scene Builder могут включать в себя компоненты, 
которые не содержат последние обновления безопасности и не рекомендуются для использования в производстве.

Это означает, что они не включают в себя последние изменения 8u40, как новый Spinner контроль.
(текстовое поле в одну строку, которая позволяет пользователю выбрать номер или значение объекта
 из упорядоченной последовательности)

Альтернативы:
Проект является открытым исходным кодом, и вы можете найти обновленный код на OpenJFX репозитории.
Вы можете загрузить проект и построить самостоятельно.

К сожалению, до будущих обновлений, альтернатив этому пока нет.
